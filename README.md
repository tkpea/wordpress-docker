# wordpress-docker
## START
$ docker-compose up

### WORDPRESS URL
http://localhost:8000

### PHP MY ADMIN
http://localhost:8001

## コンテナの中に入る
$ docker exec -i -t --user wordpress wordpress-docker_wordpress_1 bash

### WP-CLI
$ wp --version


### composer
$ composer --version

### Node.js
$ node -v

### NPM
$ npm -v

### YARN
$ yarn -v
